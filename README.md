## About this Repo



### Getting started

- Starting in the root directory, from your terminal run
- `npm install`
- `npm audit fix`
- `cd app`
- `npm install`
- `npm audit fix`
- After installing all the dependencies
- `cd ..` 
- Run `npm run dev`
- This will get the app running on your local server
- After its done compiling and loading, visit
- `http://localhost:3000/`
